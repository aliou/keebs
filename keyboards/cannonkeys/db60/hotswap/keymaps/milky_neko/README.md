# CannonKeys DB60 hotswap `milky_neko` keymap

Build target: `cannonkeys/db60/hotswap:milky_neko`.

Legend: `Trns` is transparent. `Ctrl/Esc` taps Escape and holds Control. `Ctrl/Fn2x` is Control, with double-tap for the macOS Fn/Globe shortcut. `MN_DFU` resets into DFU when held for 500ms.
