# DZTech DZ60RGB v2.1 `silent_tofu` keymap

This keymap is archived and is not part of `qmk.json` active build targets.

Legend: `Trns` is transparent. `Ctrl/Esc` taps Escape and holds Control. `Slash/Sh` taps slash and holds right Shift. `LK_DFU` resets into DFU when held for 500ms. `EE_CLR` clears EEPROM. `NK_TOGG` toggles N-key rollover.
