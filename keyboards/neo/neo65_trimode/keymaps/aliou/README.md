# NEO65 `aliou` keymap

Build target: `neo/neo65_trimode:aliou`.

Legend: `Trns` is transparent. `Ctrl/Esc` taps Escape and holds Control. `Ctrl/Fn2x` is Control, with double-tap for the macOS Fn/Globe shortcut. `Boot` enters WB32 DFU. `USB/BT1` toggles only between USB and Bluetooth slot 1. Non-physical `KC_NO` positions are omitted.
